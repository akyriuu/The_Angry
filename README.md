# The Angry — Landing Page

Landing page para a loja **The Angry**, marca de camisetas com atitude. _"Vista sua atitude."_

Site estático, responsivo, tema dark com acento laranja. Sem frameworks e sem build.

HTML, CSS e JavaScript puro (ES Modules).

## ✨ Funcionalidades

- Layout responsivo (desktop e mobile)
- Tema dark com identidade visual da marca
- Hero, Sobre, Diferenciais, Galeria e Contato
- Conteúdo centralizado em um único arquivo de configuração (`src/config.js`)
- Links diretos para Instagram e WhatsApp

## 🛠 Tecnologias

- HTML5
- CSS3 (Grid, Flexbox, variáveis CSS)
- JavaScript (Vanilla, ES Modules)
- Fontes: Bebas Neue + Inter (Google Fonts)

## 📁 Estrutura

    the-angry-landing/
    ├── index.html          # Marcação da página
    ├── styles/
    │   └── styles.css       # Estilos
    ├── src/
    │   ├── config.js        # Conteúdo do site (marca, links, contato)
    │   └── main.js          # Aplica config + interações
    └── assets/
        └── images/          # Imagens (hero, about, galeria...)
